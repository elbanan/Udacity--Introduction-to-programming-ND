Stage 0 : Getting started - Important Concepts
We learned what the internet is compased of. Internet is composed of mutliple pages written in a common language called HTML.These pages are stored on dedicated computers called servers. We use browsers installed on our computers to view pages stored on these servers (e.g. www.udacity.com).

HTML is composed of different components called elements. These elements usually have an opening tag, a closing tag and content in between. Some tags don't require closing tags (e.g. "br") and thus called void tags. HTML tags must be written in the correct syntax as computers are stupid and will not correct our mistakes based on our intentions.

It is worth mentioning that Block tags create an imaginary outline around the contents that can be customized. examples for it is the "p" tag. On the other hand, Inline tags don't create that imaginary cotainer. Examples for these are "br", "img" and "b" tags.
Stage 1 - part I : HTML
As mentioned earlier, HTML is the main building language of webpages. It can be considered as the tool that lays the foundations and parts of the new house (i.e. New webpage).
HTML has certain structure including : <html> to define that the langauge used is HTML , <head> into which title and inportant linked files (e.g. CSS) are linked and <body> which will have the main bulk of code for our webpage.

Elements within a webpage follow a tree like structure in their coding starting with larger (container) to smaller individual elements.However, all elements, including text, images, videos and links, are housed within rectangular compartments regardless of the shape of the object.

To wase the format and use afterwards, each element (or group of elements) should be identified with a unique identifier called class which will be used to refer to this element anytime. For exmple, class of div can be assigned like that : <div class = "value" >.
Stage 1 - part II : CSS
As HTML lays the foundations of the new page, title = "Cascade Style Sheet">CSS determines the look/style of the webpage elements including but not limited to : font, color, size, etc. CSS code is most commonly stored into a separate file accopmanying the website with extension .css. For CSS data to load, CSs file has to be linked to the webpage of interest by using the following code in the <head> section :

<link rel="stylesheet" type="text/css" href="file location">

Each element's style can be customized by using the unique element id "class" from HTML code using the following structure :

.element class {
style data ;
}
